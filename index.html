Pushnewsmth2mail

An desktop application to push content of some board of newsmth to mail, if the mail is kindle mail, it goes to Kindle device. Be noticed, in order to save space, the ./lib/*.jar files has been replaced with faked text file, your should replaced them back to the real jar files if you want to run it.
